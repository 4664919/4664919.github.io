覆盖率测试－测试相关－golang

2016-01-25

相关参数

名称 示例 注释
-cover -cover 表示开启覆盖率
-covermode -covermode=set 覆盖率统计模式：
set只记录是否执行过.
count统计执行次数.
原子方式atomic统计执行次数.
-coverpkg -coverpkg pkg1,pkg2,pkg3 统计覆盖测试其他包，多个以，号分隔
-coverprofile -coverprofile cover.out 数据写入到指定文件

DEMO
