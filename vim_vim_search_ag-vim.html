vim搜索插件ag.vim

之前使用vim搜索特别麻烦，vim自带的vimgrep速度慢，而且不好用。


先安装 the_silver_searcher

brew install the_silver_searcher


vim下配置：

Bundle ‘https://github.com/rking/ag.vim‘

let g:ackprg = ‘ag –nogroup –nocolor –column’



git地址传送门

https://github.com/rking/ag.vim
